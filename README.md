# Anlisis-de-Ventas-y-Rendimiento-de-Empresa-X
sistema de análisis de ventas en Excel, incluyendo preparación de datos (Power Query) e implementación de cálculos de ventas y márgenes. Creé un Dashboard interactivo con elementos clave (ventas mensuales, margen, top 10 subcategorías) para generar insights y facilitar la toma de decisiones estratégicas
